# منصة لغز اليوم — نسخة Netlify

1. فك الضغط وارفع المجلد إلى GitHub، ثم في Netlify اختر **Add new project → Import an existing project** واختر المستودع. لا تحتاج إلى إعداد Build command؛ سيقرأ Netlify الإعدادات تلقائيًا.
2. أو من جهازك، بعد فك الضغط، شغّل `npx netlify deploy --prod` داخل المجلد وسجّل الدخول عند الطلب.
3. بعد النشر، استخدم رابط الموقع لصفحة المشاركين (`index.html`) وأضف `/admin.html` لرابط لوحة التحكم.

تخزن البيانات في Netlify Blobs وتستمر بين الزيارات. لا تدعم طريقة السحب والإفلات وحدها وظائف Netlify، لذلك استخدم GitHub أو الأمر أعلاه. لا تشارك رابط لوحة التحكم علنًا؛ بإمكانك حمايته من إعدادات Netlify إذا رغبت.
